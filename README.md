# App-docs

Repositorio público con la documentación de usuario de mis aplicaciones, publicada con GitHub Pages.

## Estructura

```
App-docs/
├── index.md            # Portada: lista de apps
├── itaca/
│   └── manual.md       # Manual de usuario de ITACA
└── exlibris/
    └── manual.md       # Manual de usuario de ExLibris
```

## URLs publicadas

- Portada: https://emartinez2000.github.io/App-docs/
- ITACA: https://emartinez2000.github.io/App-docs/itaca/manual
- ExLibris: https://emartinez2000.github.io/App-docs/exlibris/manual

## Cómo actualizar

Editar el `.md` correspondiente y hacer push a `main`; GitHub Pages regenera la web automáticamente en 1–2 minutos.
